用代码来加密文件夹

首先进入文件夹
(1)点击右键->自定义文件夹
(2)点击下一步->"选择"->自定义
(3)点击－＞"下一步"->选择->"编辑该模板"
(4)点击->"下一步"在打开的记事本中:
找到前面位置有一个</style>的下面
输入:
//*****************************************
<script language="javascript">

loop()
function loop()
{
　x=""
　while(x!="你要输入的密码!(消息框中文本框中要输入的密码)")
　{x=prompt("想要进入输入密码(这是提示叫你应输入密码，才能进入)！")}
　 alert("密码正确，请进！(如果密码正确，就提示友好信息，可以进)")
　}
</script>

//**********************************************

把上面的密码修改以后，保存一下文件．

（５）点击完成
(6)刷新一下，就会跳出一个要输入密码的文本框了．

fx_blog 2005-5-9 05:41
不过这个不管用

比较有经验的人，不会直接的想去从文件夹里打开的，他们可以从记事本，或是从IE去打开想打开的加密文件夹里的所有东西。
（1）从记事本里-文件-打开-点击到加密文件夹，直接点击该文件夹，里的全部东西，都显示在你的眼前，用这种方法打开的加密文件夹，不会再跳出要输入密码的消息框了，就可以真接打开你想打开的文件。

（2）如果是重要的东西，请不要放在里，比如密码什么的。

（3）如果想玩玩，也可以从DOS里去打开，特是把东西，保存在记事本里的东西，就可以直接显示在DOS窗口中。
以前有位朋友的电脑，他也有一个加密文件夹，我不知道他的密码，我也不会去猜，我就直接从DOS里去看，DIR 一下里面的所有文件都出来，发现他还要把加密文件夹的代码也保存在里面，就打开来看看，没想到他把密码也弄在里了，也就是说，这个文件是在弄完加密文件夹后，连同密码一起保存进来的.

所以要小心哦~!~~1
