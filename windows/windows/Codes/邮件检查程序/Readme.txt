功能简介:
    每天在上班时间想看自己的邮箱的邮件都要到网页去登录检查，但一般的时候都
是没有新邮件或者早就有新邮件了。所以就做了这个简单的新邮件检查程序，它会自
己检查你的邮箱是否有了新的邮件，并且会弹出一个对话框来通知你！这样就方便多
了！
    本人提供了源代码，希望大家能增加新的功能！
