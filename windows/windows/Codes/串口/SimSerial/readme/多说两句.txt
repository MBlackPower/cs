
开始：
做这个小项目的初衷是看到很多驱动开发新手询问如何做虚拟设备的开发，其中很大一部分问题集中在虚拟串口的开发上。我从事了一段时间的驱动开发工作。做过若干小型的设备驱动程序，感觉有一些入门。正巧有一段时间不算太忙，也就斗胆开始了这个项目的开发尝试。

过程：
最初查阅了很多虚拟串口的软件。最后锁定在仿制商业软件Virtual Serial Ports Driver XP3上。通过在驱动开发网上各种相关话题的阅读学习以及对该商业软件的分析研究，确定了最终的开发思路。期间又因为各种事务中断了好几个月。

收获：
1、通过分析toaster，对总线驱动、功能驱动有了更多的了解
2、通过使用filter进行辅助工作，对filter的加载有了深入的了解
3、对串口的通信、控制机制有了一定的了解
4、熟悉了Kernel mode下的string操作

待续未完：
总算赶在春节以前完成了一个雏形。因为农历猴年春节即将来临，我也要收拾行装走一趟长线旅游。估计最近2个月没有时间进行下一步的工作。目前程序还没有进行超过2个串口的测试。还有很多需要改进的地方。趁着春节之前发布给大家，希望对新手有些许帮助，更希望各位大牛不吝赐教，多多提意见（直接发到我的信箱即可）。

最后祝大家2004年新年快乐！
